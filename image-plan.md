# Image Plan — every image slot on the site

How to use in Lovable: drag/attach the image into Lovable's chat and say e.g. "use this as the homepage hero image, alt text: …". Lovable replaces the grey placeholder.

Legend for "Source":
- **REAL** = must be a genuine photo (authority/trust — do NOT use AI/stock people here)
- **STOCK** = tasteful stock photography is fine (decoration, no authority claim)
- **BRAND** = your existing brand asset (logo etc.)

## Priority order (what to gather first)
1. **Logo + favicon** (brand) — you have these already.
2. **Real photos of Adrian** (home, about, optional on services) — the single most important set for authority. Professional headshots ideal; good natural shots fine.
3. **One OG share image** (1200×630, branded).
4. **Stock wedding photography** — hero + 6 cards + page headers. Buy/download a consistent set (same tone/colour) from one library so the site looks cohesive.
5. **Reviews badge** from reviews.co.uk.

## Stock sources (consistent, license-safe)
Unsplash / Pexels (free), or Shutterstock / iStock / Adobe Stock (paid, more choice). Pick ONE wedding-photography style and stick to it across all slots so the site feels designed, not assembled.

## Rules
- Real photo wherever a person represents YOU or your business (Adrian, clients). Never an AI/stock "fake Adrian".
- Every image gets descriptive alt text (table above) — SEO + accessibility.
- Compress to WebP, keep each under ~200KB where possible (page speed = ranking).
